# Portfolio Personal

Portafolio web interactivo desarrollado con React y Vite, presentando mis proyectos, habilidades técnicas y experiencia profesional.

## Características

- Diseño moderno y responsivo
- Animaciones y efectos interactivos
- Galería de proyectos
- Sección de habilidades técnicas
- Información de contacto y redes sociales

## Tecnologías

- React
- Vite
- Tailwind CSS
- Framer Motion

## Despliegue

Desplegado en Vercel para acceso público.
